---
description: Audita dependencias vulnerables y verifica que los tests siguen pasando
---

Ejecuta `npm audit` para encontrar paquetes instalados con vulnerabilidades.
Ejecuta `npm audit fix` para aplicar las actualizaciones seguras.
Ejecuta los tests para verificar que las actualizaciones no han roto nada.
Resume qué se actualizó y cualquier vulnerabilidad que requiera intervención manual.
